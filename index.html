Buttons

Button Colors

Define a button with class.button,. To modify buttons color use below classes, by default button style will be primary.button--success,.button--warning,.button--danger

Outlined Buttons

Define an outlined button with class.button--ouline, and no need to use.button class. To modify buttons color use below classes, by default button style will be primary.button--success,.button--warning,.button--danger

Button sizes

Button sizes are modifiers and needs to be used with.button,.button--outline,or any button type class. The modifiers are.button--xs,.button--s,.button--m,.button--lor.button--xl.The default button size is.button--m.

Icon Buttons

Add Icons to buttons by adding ionicons icon tag eg:

<button class="button--icon" type="button"> 
 <i class="ion-plus-round"></i> 
 Primary icon Button 
</button>
and use.button--icon,as the button class, Use any modifiers with buttons to change color or size

Special Buttons

Loading button
<button class="button margin-right-m is-loading" type="button">loading...</button>
Disabled button
<button class="button margin-right-m disabled" type="button">button</button>
